**Outdoor Maps**

This project was a part of spring 2026 mobile application development course in Oulu University of Applied Sciences

A simple app that combines weather data with user-submitted sightings on a map, built for hiking and outdoor exploration.

**Features**
- Interactive map with user “sightings” (locations, notes)
- Current weather data for location
- Search and filter sightings
- User specific “My Sightings” list
- Clean, minimal UI focused on usability

  
**Tech Stack**
- React Native (Expo)
- Firebase (auth + data storage)
- Maps (maanmittauslaitos)
- plant stuff (plantnet.org)
  

**How It Works**

Users can browse a map, check weather conditions, and add their own sightings. Each sighting is tied to a user ID, allowing personal tracking and optional sharing.
